Projeto 1 - Jogo

O primeiro projeto da disciplina Experiência Criativa: Navegando na Computação foi desenvolvido utilizando o ambiente Construct 3, da empresa Scirra Ltd. Este projeto, que consistiu na criação de um jogo, foi realizado em equipe, promovendo a colaboração entre os participantes e a troca de ideias criativas e técnicas. A equipe responsável pelo desenvolvimento foi composta pelos seguintes integrantes:

  • Abilio Pedro Alcantara Mota Batista
  • Felipe da Silva Cesca
  • Karen Cristini Nogueira
  • Pedro Muller Volpe
  • Samuel Garcia Pereira

O jogo desenvolvido recebeu o título Woody's Train Trouble e é inspirado no universo cinematográfico do filme Toy Story 3. Voltado para o público infantil, o jogo possui uma ambientação no velho oeste e apresenta personagens carismáticos, como Woody, Sr. Cabeça de Batata e Sr. Doutor Porcão. Trata-se de um jogo de plataforma com design visual majoritariamente produzido no Pixilart, além de elementos gráficos criados no Paint e diretamente no Construct 3. Para enriquecer a experiência do jogador, o projeto incorporou trilhas sonoras e efeitos sonoros cuidadosamente selecionados, tornando-o mais imersivo e envolvente.

Projeto 2 - Aplicativo Multimídia

O segundo projeto da disciplina Experiência Criativa: Navegando na Computação teve como base o ambiente de desenvolvimento Processing, utilizando a linguagem de programação Java. Este projeto foi concebido como uma aplicação multimídia com caráter educativo, buscando explorar conceitos de interatividade e aprendizagem. A equipe responsável pelo desenvolvimento foi formada pelos seguintes integrantes:

  • Felipe da Silva Cesca
  • Karen Cristini Nogueira

Nomeado Recycle Right, o aplicativo foi projetado para educar crianças sobre a importância da reciclagem e como incorporar essa prática em suas rotinas diárias. Por meio de elementos visuais e interativos, o projeto visa sensibilizar os jovens usuários sobre questões ambientais, estimulando a conscientização e a adoção de hábitos sustentáveis.

Projeto 3 - Desenvolvimento Web

O terceiro projeto, também parte da disciplina Experiência Criativa: Navegando na Computação , foi desenvolvido com foco em tecnologias web, utilizando o editor Visual Studio Code e as linguagens HTML, CSS e JavaScript. Este projeto representa uma evolução do segundo trabalho realizado, mantendo sua essência e objetivo, mas com uma abordagem voltada para o ambiente digital. A equipe responsável pelo desenvolvimento foi composta pelos seguintes integrantes:

  • Edgar Mateus Marcondes de Campos
  • Felipe da Silva Cesca
  • João Paulo da Silva Picoli
  • Karen Cristini Nogueira

Este projeto, também intitulado Recycle Right, foi projetado como uma ferramenta web para educar crianças sobre reciclagem, utilizando elementos visuais interativos e uma interface amigável. Através desta aplicação, espera-se ampliar o alcance e a acessibilidade das lições sobre práticas sustentáveis, demonstrando como a tecnologia pode ser usada para abordar problemas reais e promover a conscientização ambiental. Utilizou-se o banco de dados MySQL junto com XAMPP para fazer as requisições.
