Projeto 1 - Joquempo

Este projeto teve como objetivo desenvolver, em Python, um jogo interativo de Joquempo (pedra, papel e tesoura) para ser executado no terminal. A aplicação foi projetada para oferecer três modos de jogo distintos, visando aumentar a flexibilidade e o engajamento dos usuários:

  • Humano x Humano: Dois jogadores interagem diretamente no terminal, inserindo suas escolhas.
  • Humano x Computador: Um jogador compete contra o computador, que utiliza lógica programada para determinar sua jogada.
  • Computador x Computador: Dois jogadores virtuais, simulados pelo programa, competem entre si, permitindo análises e observação do comportamento do sistema.

Este projeto permitiu a aplicação de conceitos fundamentais de lógica de programação, manipulação de entrada e saída no terminal, e estruturação de código em Python. A simplicidade do terminal foi aproveitada para priorizar a eficiência e clareza do funcionamento. O projeto foi desenvolvido individualmente por:

Projeto 2 - Máquina de Bebidas

No segundo projeto, também desenvolvido em Python, o desafio consistiu na criação de uma simulação de uma máquina de bebidas com funcionalidades avançadas. A aplicação foi projetada para atender a dois tipos de usuários: clientes e administradores. Entre as funcionalidades implementadas, destacam-se:

  • Modo Cliente: Permite a seleção de bebidas e calcula automaticamente o troco com base no valor inserido pelo cliente.
  • Modo Administrador: Oferece acesso restrito a configurações do sistema, como reabastecimento de estoque, ajuste de preços e consulta ao histórico de vendas.

Este projeto explorou conceitos mais avançados de programação, como gerenciamento de fluxo condicional, operações matemáticas para cálculo de troco para o modo administrador. A interface em terminal foi utilizada de forma criativa para simular a experiência de uso de uma máquina de vendas automática. O projeto foi desenvolvido individualmente por:
